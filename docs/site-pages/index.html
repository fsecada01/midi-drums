MIDI Drums Generator

A plugin-based Python system for creating realistic drum tracks across Metal, Rock, Jazz, and Funk with 7 legendary drummer styles and Reaper DAW integration.

Get Started → See Recipes API Reference 
# Generate a complete death metal song in 3 lines
from midi_drums.api.python_api import DrumGeneratorAPI

api = DrumGeneratorAPI()
song = api.create_song("metal", "death", tempo=180, complexity=0.85)
api.save_as_midi(song, "death_metal.mid")

# Or apply a legendary drummer's signature style
pattern = api.generate_pattern("jazz", "verse", "swing")
weckl_pattern = api.apply_drummer_style(pattern, "weckl")
api.save_pattern_as_midi(weckl_pattern, "jazz_weckl.mid", tempo=130)

# Or describe what you want in plain English
pattern, info = await ai.generate_pattern_from_text(
    "heavy progressive rock with polyrhythmic fills and ghost notes",
    tempo=145, bars=4
)

# features

🎸

Multi-Genre Engine

Four fully-realized genres, each with 7 distinct styles covering the full spectrum from death metal blast beats to bebop swing.

  • Metal · Rock · Jazz · Funk
  • 7 styles per genre
  • Authentic pattern libraries
🥁

Legendary Drummers

Seven iconic playing styles implemented as composable modifications — apply any drummer to any genre pattern.

  • Bonham · Porcaro · Weckl
  • Chambers · Roeder · Dee · Hoglan
  • Signature fills included
🏗️

Song Structure

Generate complete songs with configurable section arrangements, dynamic fills, and pattern variations across bars.

  • Intro · Verse · Chorus · Bridge
  • Configurable bar counts
  • Automatic fill placement
🎛️

Reaper Integration

Export drum tracks directly to Reaper projects with automatically placed section markers at exact bar positions.

  • Creates .rpp project files
  • Section markers auto-placed
  • Works on existing projects
🤖

AI Generation

Describe the drum track you want in plain English. Supports Anthropic, OpenAI, Groq, and Cohere backends.

  • Natural language input
  • Genre/style auto-detection
  • Provider-agnostic API
🔌

Plugin Architecture

Clean SOLID architecture with strategy, builder, and factory patterns. Add new genres or drummers in ~60 lines.

  • Composable templates & mods
  • Type-safe throughout
  • Zero magic numbers
